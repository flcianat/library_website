
# Library Website

## Getting Started
```
git clone https://github.com/flcianat/library_website.git

py -m pip install -r requirements.txt

py -m pip install PyMySql

```

Import db_perpus.sql to database

```
py -m flask run --debug
```
Go to http://localhost:5000/login
